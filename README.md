# 美团优惠推广 Skill

这是一个面向美团优惠领取场景的 Skill，核心目标是帮用户更快领取专属红包、查看当日活动，并支持定时提醒，减少错过优惠的情况。

## 美团简介

美团是国内常用的本地生活服务平台之一，覆盖外卖、到店餐饮、酒店旅游、休闲娱乐、买菜、生活服务等多个消费场景。用户通常会在美团 App 中完成下单、到店消费、团购购买和优惠券使用，因此围绕“领券 + 活动 + 到店/下单转化”的体验非常关键。

## 核心亮点

| 亮点 | 说明 |
|------|------|
| 领券超便捷 | 覆盖吃喝玩乐全场景，点一下就能领，领完马上用，无需等待。 |
| 优惠早知道 | 每日热点活动一手掌握，专属链接直达，抢券快人一步。 |
| 省钱自动化 | 设置一次，每天自动提醒你领券，再也不怕错过任何福利。 |

## Skill 能做什么

这个 Skill 主要提供以下能力：

- 自动识别“领券、领红包、领优惠、美团优惠、美团红包、今日活动”等意图。
- 登录美团账号后，自动执行领券流程并返回结果。
- 同步展示当天可看的优惠活动和专属会场入口。
- 支持每日定时提醒，帮助用户持续领取优惠。
- 提供退出登录、清除设备标识、诊断排查等管理能力。

## 使用方式

### 1. 触发方式

用户可以直接输入以下类型的话术触发：

- `领券`
- `帮我领美团红包`
- `美团今天有什么优惠`
- `我要领优惠券`
- `今日活动`
- `...`（其他类似表达）

### 2. 首次使用流程

1. 首次触发时，Skill 会先展示欢迎语和协议确认内容。
2. 用户同意后，如果本地没有有效登录态，会提示输入美团绑定手机号。
3. Skill 发送验证码并完成登录校验。
4. 登录成功后自动执行领券，并返回优惠券结果或活动信息。
5. 领券完成后，会询问是否开启每日定时提醒。

### 3. 日常使用流程

- 已登录时，用户再次说“领券”即可直接走发券流程。
- 如果当天已领取过，Skill 会提示已领取，并补充今日活动信息或提醒状态。
- 如果当前暂无可领券，Skill 会返回“暂无上新”并继续保留后续提醒能力。

### 4. 常用管理指令

| 指令类型 | 示例 |
|------|------|
| 退出登录 | `退出登录`、`切换账号` |
| 清除设备标识 | `清除设备标识`、`重置设备` |
| 定时提醒 | `取消提醒`、`改成8点提醒我`、`几点提醒我` |
| 诊断排查 | `诊断` |

## 使用规则说明

- 严格按照 `Step 0 -> Step 5` 的顺序执行，不跳步。
- 每次用户触发领券，都必须实际调用脚本执行，不能凭记忆直接回复。
- 手机号、验证码、Token 等敏感信息只保存在本地，不上传到第三方。
- 出错时按 Skill 预设话术处理，不暴露技术细节。
